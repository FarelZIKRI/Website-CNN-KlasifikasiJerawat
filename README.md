# AcneAI - Klasifikasi Jerawat dengan CNN

Sistem klasifikasi tipe jerawat menggunakan Deep Learning Convolutional Neural Network (CNN) dengan arsitektur EfficientNet.

## Deskripsi

Website ini merupakan landing page interaktif untuk sistem klasifikasi jerawat yang dapat mengidentifikasi 3 tipe jerawat:

- **Jerawat Ringan** - Komedo hitam, komedo putih, papula kecil
- **Jerawat Sedang** - Papula meradang, pustula, nodul kecil
- **Jerawat Parah** - Nodul besar, kista berisi nanah

## Fitur

- Deteksi cepat menggunakan model deep learning
- Akurasi tinggi dengan arsitektur EfficientNet
- Antarmuka drag & drop untuk upload gambar
- Laporan detail dengan tingkat kepercayaan prediksi
- Desain responsif dan modern

## Teknologi

- **Framework:** TensorFlow & Keras
- **Arsitektur:** EfficientNet
- **Frontend:** HTML5, CSS3, JavaScript
- **Font:** Inter (Google Fonts)
